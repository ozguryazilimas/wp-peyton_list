####   Version 5.0.0 (2026-02-16)

### New Features
- Complete dashboard redesign with improved user experience and interface
- Reorganized settings structure that groups related options together, making configuration more straightforward
- Adds a new generation method for YouTube and Vimeo videos when using the **Find in post** generation option
- Adds support for cancelling the process of deletion & generation of thumbnails
- Adds support for using Google Fonts to generate the featured image from the title (Pro)
- Adds gradient color options for backgrounds (Pro)

### Enhancements
- Removed support for outdated external APIs
- Improved preview mode to happen live when plugin settings are changed
- Enhanced the Delete Featured Images functionality
- Ensures plugin is fully translation-ready
- Improves user experience in the Posts list page, updating the featured image for each post when its set
- Dashboard now shows realtime progress of the generation & deletion processes
- Migrates license management to the Themeisle store (Pro)
- Improves UI for the media modal integration & Media dashboard page (Pro)

### Bug Fixes
- Fixed multiple PHP errors
- Fixed an issue where logs couldnt be downloaded
- Fixed an issue where featured images were applied to unselected post types
- Resolved a critical error occurring during the export logs process
- Fixed an issue where ampersands (&) in titles appeared as HTML entities (&amp) in thumbnails
- Fixes cache issue where deleting and regenerating thumbnails would show the old cached thumbnail instead of the newly generated one
